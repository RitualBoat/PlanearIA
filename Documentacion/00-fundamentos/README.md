Reglas vigentes:

- PlanearIA es una suite docente conectada.
- Office Docente une documentos, hojas, listas y entregables escolares.
- Classroom organiza, asigna y da seguimiento.
- IA propone; el docente decide y confirma acciones importantes.
- Offline-first, `src/sync`, `userId`, `aiGateway` y presupuesto bajo son restricciones base.
- UI visible se valida con evidencia visual, no solo con tests automaticos.
