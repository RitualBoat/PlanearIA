# Planeaciones Ground Truth

> Estado: indice minimo. Plantillas, capturas y documentos completos viven en respaldo externo.
> Uso: recordar que Office Docente/NotasPLAN requiere ground truth real antes de paridad alta.
> No usar para: reconstruir plantillas desde memoria o inventar UI.

Para cambios visuales de planeaciones, solicitar el respaldo externo, Figma o una referencia nueva aprobada. Sin ground truth, el change debe registrarlo como riesgo o bloqueo.
