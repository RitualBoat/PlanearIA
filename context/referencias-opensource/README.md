# Referencias Open Source Externalizadas

> Estado: indice minimo / no ejecutable.
> Uso: recordar que las referencias completas existen fuera del repo en respaldo del usuario.
> No usar para: copiar codigo, importar archivos o pedir a CodeGraph que razone sobre fuentes externas.

Si un change necesita una referencia open source, solicitar el respaldo externo o clonar la fuente fuera del repo principal. Registrar URL, licencia y rutas relevantes en `design.md` antes de usarla como inspiracion.
