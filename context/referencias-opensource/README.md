2. Qué usar de cada repo

openwordwriter/openwordwriter: útil solo como referencia conceptual de procesador de texto. Es C++/AbiWord y GPL-2.0, así que no conviene copiar código ni meterlo al proyecto. Fuente: GitHub lo describe como fork de AbiWord y licencia GPL-2.0.

dromara/yft-design: el más útil para selector tipo Canva/Genially. Usa Vue3, Fabric.js y MIT; tiene ideas buenas de canvas, plantillas, reglas, zoom, guías, export PDF/SVG/imagen y edición de páginas.

fancyboi999/Loomic: útil para IA contextual sobre canvas. Usa Next.js, Fastify, LangGraph, Supabase, Excalidraw. Inspiración para acciones IA con contexto visual, pero demasiado grande para copiar ahora.

11cafe/jaaz: útil para filosofía local-first/privacidad/IA multimodal y flujo tipo “Magic Canva”. Bueno para futuro módulo IA/canvas, no para hotfix actual.

devjaw/Rishah: útil por UX minimalista/offline con tldraw/Tauri. Inspiración visual y de experiencia, no tanto para Planeaciones.
